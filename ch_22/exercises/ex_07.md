# Solution

`n = scanf("%d%f%d", &i, &x, &j);`

1. `10•20•30¤`
   1. i = 10, 
   2. x = 20, 
   3. j = 30,
   4. n = 3
2. `1.0•2.0•3.0¤`
   1. i = 1, 
   2. x = 0.0, 
   3. j = 2
   4. n = 3
3. `0.1•0.2•0.3¤`
   1. i = 0, 
   2. x = 0.1, 
   3. j = 0
   4. n = 3
4. `.1•.2•.3¤`
   1. i = undefined, 
   2. x = undefined, 
   3. j = undefined,
   4. n = 0
